librdf - Included Packages

I AM NOT A LAWYER but I've included my evaluation of the licensing issues with using the included software with librdf.

Expat - XML parser

http://www.jclark.com/xml/expat.html

A C library implementing a fully conforming, non validating XML 1.0 parser. Resources: Home, FAQs, Source

expat is licensed under the GPL and MPL licenses and there are compatible combinations with librdf:

librdf / expat compatibility chart
  expat
GPL MPL
librdfGPL/LGPL YES NO
BSD YES YES

libxml - XML parser for Gnome (a.k.a gnome-xml)

http://xmlsoft.org/

A C library for providing DOM and SAX like interfaces to tag-based structed documents/data in XML. Resources: Home, FAQs, Documentation, Source, CVS repository

libxml is licensed under the LGPL and W3C IPR licenses and there are compatible combinations with librdf:

librdf / libxml compatibility chart
  libxml
LGPL W3C IPR
librdfGPL/LGPL YES YES
BSD YES YES
